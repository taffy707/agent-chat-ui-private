## About Vertex AI Search

Vertex AI Search is a fully-managed platform powered by large language models that enables developers to build AI-enabled search and recommendation experiences. It leverages Google's expertise in information retrieval and natural language processing.

## Key Features

1. **Automatic Text Extraction** - Extracts text from PDF, DOCX, and HTML files
2. **Intelligent Chunking** - Optimally chunks documents for better search results  
3. **Semantic Embeddings** - Generates embeddings for semantic search
4. **Hybrid Search** - Combines keyword and semantic search
5. **LLM-Powered Summaries** - Provides AI-generated summaries with citations

## How It Works

When you upload a document:
- The file is stored in Google Cloud Storage
- Vertex AI Search automatically processes it
- Text is extracted and chunked
- Embeddings are generated
- The document is indexed for search

## Benefits

- No manual processing required
- Google-quality search results
- Scales automatically
- Built-in LLM integration
- Enterprise-grade security
